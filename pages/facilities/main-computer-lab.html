Main Computer Lab

The main computer lab consists of 60 high end computers, which are equipped with both windows and linux operating systems. This lab also consists of all the multimedia facilities so lab sessions can be done interactively. This facility is used for lab classes of many areas such as basic programming, Artificial Intelligence, Data Mining and Operating Systems.
